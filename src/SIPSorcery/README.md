# SIPSorcery package README

This file is the per-folder README for the **SIPSorcery** package's source
directory. The full project README -- the same one rendered on the
[NuGet package page][nuget] and on the repository home page -- lives at the
repo root and is what gets shipped inside the `.nupkg`. Read it there:

> **[Go to the main SIPSorcery README][main-readme]**

Why this file exists at all: GitHub renders a folder README inline when
visitors browse a directory. Without this stub, the SIPSorcery folder
would have no description and visitors who deep-linked here from
nuget.org's *Project website* link would land on a bare file listing.
The actual package documentation, install instructions, supported
protocols, getting-started for VoIP, and getting-started for WebRTC
are all in the root README to avoid duplication.

[main-readme]: https://github.com/sipsorcery-org/sipsorcery/blob/master/README.md
[nuget]: https://www.nuget.org/packages/SIPSorcery
